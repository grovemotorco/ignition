## Prerequisites

- **[Bun](https://bun.sh)** runtime (v1.3 or later)
- **SSH client** installed on your machine (the `ssh` and `scp` binaries)
- **SSH access** to your target hosts (key-based authentication recommended)

Ignition targets **Debian/Ubuntu** hosts. The `apt` and `service` resources assume `apt-get` and `systemd`.

## Install from source

```bash
git clone https://github.com/grovemotorco/ignition.git
cd ignition
bun install
bun run scripts/install.ts
```

This links the `ignition` binary globally. Verify with:

```bash
ignition --version
```

## Run from source

You can also run Ignition directly without installing globally:

```bash
bun run src/cli.ts run <recipe.ts> <target>
```

## Build a standalone executable

```bash
bun run build:executable
```

This produces a single-file binary at `dist/ignition` that can be copied to machines of the same OS/architecture without requiring Bun to be installed.
